## v3.0.1 写作场景素材推荐

### 功能描述
在Stage 3编辑器中，根据用户实时输入内容，智能推荐相关素材。

### 验收标准
- [ ] 输入10个字符以上触发推荐
- [ ] 推荐响应<500ms
- [ ] 推荐准确率>70%
- [ ] 一键引用功能正常

### 技术实现
- 防抖处理500ms
- 关键词提取（长度>=4）
- 多维度匹配算法（标签/标题/内容）

---

## v3.0.2 素材引用统计

### 功能描述
记录和展示素材的使用情况，帮助识别高价值素材。

### 验收标准
- [ ] 引用计数准确无误
- [ ] 使用历史可追溯
- [ ] 热门素材Top10页面正常
- [ ] 报表生成<2s

### 技术实现
- asset_quotes表记录引用
- GET /api/assets/popular接口
- AssetDetail页面引用统计Tab

---

## v3.0.3 智能标签补全

### 功能描述
自动从素材标题和内容中提取关键词作为标签。

### 验收标准
- [ ] 标签准确率>80%
- [ ] 支持用户修正
- [ ] 批量打标功能正常
- [ ] 处理时间<1s

### 技术实现
- TF-IDF算法提取关键词
- 停用词过滤
- 关键短语提取
