# 专家库体系 v5.0 产品需求文档

**版本**: v5.0

---

## 1. 版本概述

### 1.1 目标

构建覆盖10+专业领域的专家库体系，每个领域配置5+位具有鲜明人设、核心思想体系和成功实践经验的专家角色，深度赋能蓝军评审和观点洞察环节，提升内容生产的深度和差异化竞争力。

### 1.2 核心价值

- **专业性提升**: 多维度专家视角确保内容专业深度
- **差异化保障**: 不同专家核心思想的碰撞产生独特洞察
- **效率优化**: 预置专家模板减少人工配置成本
- **质量可控**: 基于成功实践的方法论确保评审质量

#### E01-01 刘明远 | 周期派宏观分析师

**人设背景**:

> "当前处于库存周期底部与产能周期高位的叠加期，这意味着复苏力度有限但结构机会突出。关注先进制造和出口链的补库机会。"

## 4. 专家库与蓝军评审整合

### 4.1 评审匹配机制

```typescript
interface ExpertMatcher {
  // 任务主题匹配
  matchByTopic(topic: string, experts: Expert[]): Expert[];

  // 行业属性匹配
  matchByIndustry(industry: string, experts: Expert[]): Expert[];

  // 评审维度匹配
  matchByDimension(
    dimension: 'accuracy' | 'logic' | 'expertise' | 'readability',
    experts: Expert[]
  ): Expert[];

  // 综合匹配
  findBestExperts(task: Task, count: number): Expert[];
}
```

### 4.2 动态评审流程

```
任务创建
    ↓
主题分析 → 自动匹配领域专家(2-3位)
    ↓
蓝军评审 → 领域专家深度评审 + 通用专家(事实/逻辑/读者)交叉评审
    ↓
观点洞察 → 专家核心思想注入，生成差异化观点
    ↓
质量评估 → 基于专家评审结果综合评分
```

### 4.3 评审维度映射

| 蓝军评审维度 | 对应专家类型 | 评审重点 |
|-------------|-------------|---------|
| 事实核查 | 各领域事实核查员 | 数据准确性、来源可靠性 |

| 可读性 | 读者代表 | 表达清晰度、受众匹配度 |

## 8. 附录

### 8.1 专家数据模型

```typescript
interface Expert {
  id: string;
  name: string;
  code: string; // 如 "E01-01"

  // 基础信息
  profile: {
    title: string;
    background: string;
    personality: string;
    avatar?: string;
  };

  // 领域归属
  domain: {
    primary: string;
    secondary?: string[];
    tags: string[];
  };

  // 核心思想
  philosophy: {
    core: string[]; // 核心观点列表
    quotes: string[]; // 标志性言论
  };

  // 成功实践
  achievements: {
    title: string;
    description: string;
    date: string;
    impact: string;
  }[];

  // 评审能力
  reviewCapabilities: {
    dimensions: ('accuracy' | 'logic' | 'expertise' | 'readability')[];
    specialties: string[];
    weight: number; // 评审权重
  };

  // 状态
  status: 'active' | 'inactive';
  performance: {
    totalReviews: number;
    accuracy: number;
    avgResponseTime: number;
  };
}
```

### 8.2 评审意见模板

基于不同专家类型，预置差异化的评审意见模板：

**周期派宏观分析师**:
- "从周期位置看，当前处于[周期阶段]，这意味着[影响]..."
- "与历史周期对比，本次的特殊性在于[差异点]..."

**政策解读专家**:
- "该政策的真实意图是[意图]，执行层面需要关注[风险点]..."

- "从架构演进角度，当前设计在[场景]下可能存在[风险]..."
