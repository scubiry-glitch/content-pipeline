**用户故事**: 作为内容创作者，在编辑器中输入时，系统能自动推荐相关素材。

**技术实现**:
```typescript
const calculateRelevance = (asset: Asset, keywords: string[]): number => {

    '芯片': ['半导体', '集成电路'],
  };
  // 多维度匹配算法
};
```

**验收标准**:
- [ ] 输入10个字符以上触发推荐

- [ ] 推荐准确率>70%
- [ ] 一键引用功能正常

### 2.3 智能标签补全 (v3.0.3)

**用户故事**: 作为素材上传者，我希望系统自动为素材打上合适的标签。

**提取规则**:

**验收标准**:
- [ ] 标签准确率>80%
- [ ] 支持用户修正
- [ ] 批量处理正常
