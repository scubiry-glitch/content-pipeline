## 2. 迭代计划

### Iteration 3.3.1: 研报智能匹配增强
**目标**: ReportMatcher算法前端展示优化
**验收标准**:
- [ ] 匹配结果置信度可视化
- [ ] 支持手动调整匹配关系
- [ ] 一键创建关联任务
- [ ] 匹配历史记录

### Iteration 3.3.2: 任务与素材自动关联
**目标**: 任务自动识别并推荐相关素材
**验收标准**:
- [ ] 任务创建时自动搜索相关素材
- [ ] 素材推荐弹窗展示
- [ ] 一键添加素材到任务
- [ ] 任务详情页显示关联素材

### Iteration 3.3.3: 热点与任务智能推荐
**目标**: 热点内容智能推荐创建任务
**验收标准**:
- [ ] 热点详情页显示"创建任务"按钮
- [ ] 自动填充热点标题到任务主题
- [ ] 推荐相关研报和素材
- [ ] 热度趋势与任务关联

### Iteration 3.3.4: 素材引用统计
**目标**: Assets增加引用统计功能
**验收标准**:
- [ ] 显示素材被哪些任务引用
- [ ] 引用次数统计
- [ ] 素材影响力评分
- [ ] 热门素材排行榜

### Iteration 3.3.5: 智能预警面板
**目标**: Dashboard增加智能预警
**验收标准**:
- [ ] 任务逾期预警
- [ ] 热点爆发预警
- [ ] 素材质量下降预警
- [ ] 预警通知设置

### Iteration 3.3.6: 全局搜索(Command+K)
**目标**: 快捷键唤起全局搜索
**验收标准**:
- [ ] Command/Ctrl+K唤起搜索
- [ ] 支持任务/素材/研报/热点搜索
- [ ] 搜索结果分类展示
- [ ] 键盘导航支持
