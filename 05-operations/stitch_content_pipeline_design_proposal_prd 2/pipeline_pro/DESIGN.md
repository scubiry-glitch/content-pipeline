# Design System Document

This document outlines the core principles and specifications of our design system.

## Theme

### Color Palette

Our brand utilizes a vibrant and modern color palette:

*   **Primary Color:** `#1A73E8` (A vibrant blue, used for key interactive elements and primary actions).
*   **Secondary Color:** `#5F6368` (A sophisticated gray, supporting less prominent UI elements and secondary actions).
*   **Tertiary Color:** `#E8710A` (A warm orange accent, used for highlights, badges, and decorative elements).
*   **Neutral Color:** `#F8F9FA` (A light, clean off-white, forming the base for backgrounds and surfaces).

The overall `color_mode` is set to `light`.

### Typography

We employ a clear and legible typographic system:

*   **Headlines:** `Manrope` (A modern, geometric sans-serif for striking headlines).
*   **Body Text:** `Inter` (A versatile and highly readable sans-serif for general body content).
*   **Labels:** `Inter` (Consistent with body text for clear and functional labels).

### Shape and Form

Our UI elements are designed with:

*   **Roundedness:** `1` (Subtle rounded corners, providing a friendly yet clean aesthetic).

### Spacing

We implement a `normal` level of whitespace:

*   **Spacing:** `2` (Normal spacing, balancing content density with readability and visual comfort).
