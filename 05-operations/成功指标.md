## 1. 北极星指标

### 1.1 定义

**内容产出稳定性指数** = 实际产出期数 / 计划产出期数 × 100%

**目标值**: ≥ 90% (即双周更中最多跳票1期)

---

## 2. 核心指标矩阵

### 2.1 质量维度

| 指标 | 定义 | 现状 | 目标 | 监控频率 |
|------|------|------|------|---------|
| **蓝军评审通过率** | 一次通过评审的内容占比 | 人工把控 | 100% | 每期 |
| **内容准确性** | 无事实错误的内容占比 | - | 100% | 每期 |
| **读者满意度** | 好评率(4-5星) | - | ≥ 90% | 每月 |

### 2.2 效率维度

| 指标 | 定义 | 现状 | 目标 | 监控频率 |
|------|------|------|------|---------|
| **单期生产时间** | 选题到发布总耗时 | 数天 | < 2天 | 每期 |
| **自动化占比** | 无需人工干预的环节占比 | 0% | > 70% | 每月 |
| **研究耗时** | 执行研究到摘要生成时间 | - | < 30分钟 | 每期 |
| **写作耗时** | 大纲到初稿生成时间 | - | < 10分钟 | 每期 |

### 2.3 产出维度

| 指标 | 定义 | 现状 | 目标 | 监控频率 |
|------|------|------|------|---------|
| **更新频率** | 每月发布期数 | 不定 | 2期/月 | 每月 |
| **内容复用率** | 引用历史研报的内容占比 | 低 | > 50% | 每月 |
| **多态产出数** | 单期内容的多平台版本数 | 1 | ≥ 3 | 每期 |

### 2.4 影响力维度

| 指标 | 定义 | 现状 | 目标 | 监控频率 |
|------|------|------|------|---------|
| **平均阅读量** | 单期内容平均阅读数 | - | 5000+ | 每月 |
| **互动率** | (点赞+评论+分享)/阅读量 | - | > 5% | 每月 |
| **粉丝增长** | 月均新增关注数 | - | > 200 | 每月 |
| **转化率** | 阅读→关注转化率 | - | > 2% | 每月 |

## 4. 指标计算方式

### 4.1 蓝军评审通过率

```
通过率 = (一次通过的内容数 / 总提交评审的内容数) × 100%
```

### 4.2 自动化占比

```
自动化占比 = (自动化环节耗时 / 总生产时间) × 100%

自动化环节:
- 研究执行
- 研究摘要生成
- 大纲生成
- 初稿生成
- AI蓝军评审

人工环节:
- 选题决策
- 大纲审核
- 内容编辑
- 终审确认
```

### 4.3 内容复用率

```
复用率 = (引用历史研报的内容数 / 总发布内容数) × 100%
```

### 4.4 互动率

```
互动率 = (点赞数 + 评论数 + 分享数) / 阅读量 × 100%
```

## 5. 预警机制

### 5.1 红色预警 (立即处理)

| 触发条件 | 通知方式 | 处理要求 |
|---------|---------|---------|
| 蓝军评分 < 60 | 系统通知 | 24小时内重写 |
| 发布失败 | 系统通知 | 立即手动补发 |
| 系统故障 > 1小时 | 系统通知 | 立即联系开发 |

### 5.2 黄色预警 (关注)

| 触发条件 | 通知方式 | 处理要求 |
|---------|---------|---------|
| 单期耗时 > 3天 | 周报提及 | 分析原因，下期改进 |
| 阅读量较上期下降 > 30% | 周报提及 | 复盘选题策略 |
| 自动化占比 < 50% | 周报提及 | 优化流程 |
