## 版本概览

v3.7 是内容质量输入体系的重要里程碑，完成了多主题系统与品牌定制功能。至此，v3.x 系列（v3.2-v3.7）全部功能已完成交付。

## 功能清单

### v3.2 性能优化与用户体验 ✅
- 路由懒加载优化 - 首屏加载 < 2s
- 数据缓存机制 (SWR) - 5分钟缓存+离线检测
- 任务列表虚拟滚动 - 支持1000+任务
- 图片懒加载 - Intersection Observer
- 研报PDF预览 - 页码导航+缩放

### v3.5 AI增强与智能分析 ✅

## 技术架构

### 前端技术栈
- React 18 + TypeScript
- React Router (懒加载优化)
- SWR (数据缓存)
- CSS Variables (主题系统)

### 核心组件
- 主题系统: `src/themes/`
- 错误边界: `src/components/ErrorBoundary.tsx`
- 骨架屏: `src/components/Skeleton.tsx`
- 快捷键: `src/hooks/useKeyboardShortcuts.ts`

### 快捷键列表
| 快捷键 | 功能 |
|--------|------|
| ⌘/Ctrl + K | 打开全局搜索 |
| ⌘/Ctrl + Shift + T | 打开主题切换 |
| Shift + ? | 显示快捷键帮助 |
| Esc | 关闭弹窗/返回 |
