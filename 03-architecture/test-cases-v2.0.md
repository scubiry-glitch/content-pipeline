## 测试用例清单

### TC-001: 版本对比面板显示

**目的**: 验证版本对比面板正确显示

**测试步骤**:
1. 打开任务详情页
2. 切换到"文稿生成"标签
3. 观察版本历史面板

**测试数据**:
```json
{
  "versions": [
    {
      "id": "v1",
      "task_id": "task-001",
      "version": 1,
      "content": "初稿内容...",
      "change_summary": "初始版本生成",
      "created_at": "2026-03-17T10:00:00Z"
    },
    {
      "id": "v2",
      "task_id": "task-001",
      "version": 2,
      "content": "修改后内容...",
      "change_summary": "根据评审意见修改",
      "created_at": "2026-03-17T11:00:00Z"
    }
  ],
  "writing_data": {
    "version": 2
  }
}
```

**预期结果**:
- [x] 版本列表按版本号倒序排列
- [x] 当前版本显示"当前"标签
- [x] 最新版本显示"最新"标签
- [x] 显示版本创建时间
- [x] 显示版本变更摘要

**实际结果**: 待验证
