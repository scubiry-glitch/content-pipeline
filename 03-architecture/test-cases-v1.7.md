### TC-001: 数据质量评分显示

**目的**: 验证数据质量评分正确计算和显示

**测试步骤**:
1. 打开任务详情页
2. 切换到"深度研究"标签
3. 观察数据清洗面板

**测试数据**:
```json
{
  "annotations": [
    { "id": "1", "credibility": { "overall": 0.95 } },
    { "id": "2", "credibility": { "overall": 0.85 } },
    { "id": "3", "credibility": { "overall": 0.45 } }
  ]
}
```

**预期结果**:
- [x] 质量分数正确计算（高质量数据占比）
- [x] 分数颜色正确（>=80绿色，>=60黄色，<60红色）
- [x] 统计数据正确（总数据、重复、低质量、不完整）

**实际结果**: 待验证
