# 不动产财富管理信托 (上海信托方案)

上海信托正在探索的财富管理信托方案: 把房产登记到客户名下信托, 信托做所有权与租赁权剥离, 把租赁权折扣卖给银行/金融机构, 变现资金用于私行财富管理或养老等支付。
