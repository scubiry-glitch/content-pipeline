## 项目二：南京浦口·城市星瀚

### 项目画像

| 维度 | 内容 |
|------|------|
| **产权方** | 南京浦口区（国企/政府） |
| **项目定位** | 国际社区品质长租公寓 |
| **区位** | 南京江北中央商务区商圈 |
| **合作模式** | 委托运营（非包干） |
| **合作周期** | 5+5年 |
| **服务费** | 基础5% + 激励0~2%（按出租率阶梯） |
| **核心挑战** | 供需反转、房多客少、租金承压 |

### 合作模式对比

| 维度 | 杭州星辉时光城 | 南京城市星瀚 |
|------|--------------|------------|
| **模式** | 包干委托运营 | 委托运营（实报实销） |
| **周期** | 4个月+1年+1年 | 5+5年 |
| **收费** | 爬坡费90万+稳定期115万 | 营收5%+激励0~2% |
| **成本承担** | 贝壳承担人员+三方+物料 | 甲方承担大物业+能源+维保；贝壳承担人员+营销 |
| **风险** | 贝壳承担出租率风险（有罚则） | 甲方承担更多成本风险 |
| **关系** | 甲方首个经营类项目 | 长期战略合作 |

### 贝壳核心优势（项目方案原文提炼）

1. **平台级流量**：4000万+月活，500万日均租房曝光，3000万+年用户访问
2. **链家联动去化**：全国3.5万+门店，30万+经纪人；自营公寓链家成交占比最高30%
3. **精准线上投放**：脱离空间限制，按工作地精准展位投放
4. **自研租务系统**：壳劲租系统+资金通道，降低系统费用
5. **AI智能匹配**：客源联盟平台，站内+站外租客+企微+ACN全链路

### 经济效益

**贝壳收费结构**（减免后）：

| 收费项 | 费率 | 说明 |
|--------|------|------|
| 基础服务费 | 5% | 含中台摊销0.8%+交付验收0.5%+定价调价0.5%+运营招租2.2% |
| 品牌费 | 1% | 冠名费（原2%，减免平台使用费/宣广费/系统费共2%） |
| 激励服务费 | 0~2% | 出租率≤90%不收，90-92%收1%，92-95%收2% |

**关键**：本次合作**减免平台使用费+宣广费+系统费共2%**，体现贝壳以项目为战略入口的定价策略

### 社会效益

1. **G端品牌发声**：贝壳租房"新青年计划"获住建部、苏州/南京市委市政府、江苏省房协官方报道
2. **保租房政府平台合作**：贝壳租房成都公司已成为成都保租房政府官方平台合作伙伴
3. **行业痛点回应**：项目方案直接回应4大租客痛点——租期不确定、缺乏服务、品质不足、个性化未满足
4. **服务者职业化**：项目方案明确指出行业痛点——"经纪人平均从业时间仅8个月，月度流失率10%"——这正是REFP认证体系要解决的核心问题

### 好房子样板价值

**最大优势：5+5年长期合作+委托运营模式——天然的好房子评级验证场景**

- 委托运营模式下贝壳负责日常运营和品质管理，可以直接将好房子标准嵌入服务流程
- 5+5年周期足够长，可以做**好房子标准的纵向验证**（入住1年vs3年vs5年的品质变化）
- 激励服务费与出租率挂钩，天然验证"好房子→高出租率→高收益"的逻辑链

| 维度 | 杭州星辉时光城 | 南京城市星瀚 |

| **项目定位** | 品质集中式公寓（贝壳海盐公寓品牌） |
| **区位** | 闵行区合川路3111号，古美核心区域，临近漕河泾开发区 |

| **房间数** | 200间（每层25户，套房2套+明房23套） |
| **单房面积** | 不低于23㎡（全明间） |
| **竣工时间** | 2015年10月 |
| **合作模式** | **FAM模式**（美租集中式，贝壳负责装修+运营） |
| **合作周期** | 5+5年 |

### 经济效益

**装改投入**：1816万（单房90,818元/间，单平2,269元/平）

| 装修科目 | 金额 | 说明 |

**经营推演**：

| 指标 | 数值 |
|------|------|
| 房均价 | 4,160元/间/月 |
| 年营收 | 918万 |
| 年毛利 | 817万 |
| 装改资金回收周期 | 约26个月（开业起） |
| 装改资金规模 | 约为5年合作期营收的40% |

**市场对标**：
- 金虹桥板块：散租一居室在租均价6,000元/月（5,500~7,000元区间）
- 金汇板块：散租一居室在租均价5,200元/月（4,800~5,700元区间）

### FAM模式核心机制

```
产权人/投资人 ←→ 贝壳公寓 ←→ 信托计划
                    ↕              ↕
               装修方          共管账户(理房通)
                    ↕              ↕
                 租户      ←→  按施工进展分阶段购交易应收账款
```

- 贝壳输出产品标准+装修计划
- 信托计划按施工进展分阶段购买应收账款
- 资金通过理房通共管账户流转
- 产权人委托管理+代扣协议

### 社会效益

1. **城市更新典范**：2015年竣工的商业楼宇转型为品质长租公寓，盘活存量商办资产
2. **虹桥商务区配套**：为漕河泾科技园区、虹桥枢纽周边的就业人群提供品质租住
3. **集体经济组织示范**：产权方为闵行区虹桥镇集体经济组织，是**集体资产保值增值**的示范案例

### 好房子样板价值

**最大优势：FAM模式+高装改投入——验证"好房子标准→装修溢价→金融闭环"**

- 9万/间的装改投入，可以直接按好房子标准设计落地
- FAM模式将装修+运营+金融（信托）串联，验证铁三角中"工具→方法论"的闭环
- 4,160元/月的定价 vs 市场5,200~6,000元——有租金溢价验证空间
- 26个月回收周期的金融模型，可直接为REFA认证中"ROI=CapM(after)-CapM(before)"提供实证
